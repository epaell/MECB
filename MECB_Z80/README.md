# MECB_Z80
An implementation of a Z80 board for the MECB system. The hardware is based on designs available
for the RC2014 system. In particular it implements the 512 KB RAM and ROM paging system 
(https://rc2014.co.uk/modules/512k-rom-512k-ram-module/) that also works nicely with Wayne
Warthen's RomWBW software (https://github.com/wwarthen/RomWBW). The schematic, PCB and GAL firmware
is available for v1.0 of the board.

The board has been tested up to 10 MHz.
